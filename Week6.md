# 학습 내용 정리

## 48. 워크시트 서식(2)

<!-- 워크시트에 관해 본 강의에서 알게 된 점을 적어주세요 -->
- 글꼴, 색상, 눈금선, 머리글 정렬 등의 서식을 조정할 수 있음<br>
➡️ 차트 제목, 눈금선 등을 제거하고 주요 수치를 강조하여 시각적 서식을 조정함<br>

## 49강. 대시보드패널

<!-- 대시보드패널 강의에서 알게 된 점을 적어주세요. -->
- 시트 목록, 개체(Container), 필터 등이 위치하고, 사용자 드래그 앤 드롭으로 구성 요소를 배치 가능<br>
➡️ Top10, Sales by Year을 패널에 배치하여 구성함<br>


## 50. 대시보드 구성방식

<!-- 알게 된 점을 적고, 아래 질문에 답해보세요 :) -->

> **🧞‍♀️ 부동과 바둑판식 방식을 차이를 중점으로 기술해보세요**

| 구분    | 부동(Floating)        | 바둑판식(Tiled)           |
| ----- | ------------------- | --------------------- |
| 배치 방식 | 자유롭게 겹치거나 위치 지정 가능  | 정해진 그리드에 맞춰 자동 배치     |
| 위치 지정 | X, Y 좌표와 크기를 수동 설정  | 상하좌우로 맞물리게 배치됨        |
| 활용 사례 | 배너, 이미지, 겹치는 레이어 효과 | 반응형 레이아웃, 정렬이 필요한 보고서 |
| 장점    | 디자인 유연성 높음          | 일관된 구조 유지             |
| 단점    | 정렬과 간격 맞추기 어려움      | 레이아웃 유연성 부족           |



## 51. 대시보드 컨테이너
- 여러 시트를 묶어서 배치 정렬을 쉽게 해주는 도구
- 수평(Horizontal) / 수직(Vertical) 컨테이너로 나뉘며, 내부에 시트를 넣으면 자동으로 정렬<br>
➡️ 상단의 KPI 수치나 여러 Top10 차트들을 컨테이너로 구성해 간격과 정렬을 맞춘<br>

## 52. 레이아웃 패널
- 각 요소(시트, 컨테이너 등)의 위치, 크기, 여백, 테두리 등을 설정 영역<br>
➡️ 전체 여백과 간격이 일정하게 유지되어 레이아웃 패널로 조정했음<br>

## 53. 필터 동작

<!-- 필터 동작에 대해 알게 된 점을 적어주세요 -->
- 데이터 범위를 제한하는 역할
- 선택된 필터값에 따라 관련된 시각화들이 함께 변경<br>
➡️ 상단의 "Zone Sales" 드롭다운 필터 : 특정 지역(EU, JP, NA 등)의 판매량만 보여주며, 하단의 모든 차트에 연동되어 동작<br>

## 54. 대시보드 하이라이터 동작

<!-- 하이라이터에 대해 알게 된 점을 적어주세요 -->
- 대시보드에서 특정 값(장르, 플랫폼 등)을 강조해주는 기능<br>
➡️ 장르(Genre) 또는 플랫폼 선택 시, 해당 요소가 강조되어 다른 항목과 시각적으로 구분

## 55. 대시보드 URL

<!-- URL에 대해 알게 된 점을 적어주세요 -->
- 대시보드를 웹 링크(URL) 형태로 공유하거나, 필터값 등을 포함한 URL 매개변수로 사용자 맞춤형 뷰를 제공
- URL 공유 시 특정 필터 값을 적용하여 시작 페이지를 커스터마이징할 수 있대요


## 56. 대시보드 시트에 이동 동작

<!-- 대시보드 시트에 이동에 대해 알게 된 점을 적어주세요!-->
- 대시보드 내 특정 요소(바 차트 클릭 등)를 클릭하면 해당 시트 또는 상세 페이지로 이동 가능
- 사용자 인터랙션에 기반한 탐색 구조를 제공한대요


## 57. 매개변수 변경 동작

<!-- 매개변수 변경 동작에 대해 알게 된 점을 적어주세요!-->
- 매개변수(Parameter)는 사용자 정의 입력값을 받아서 계산된 필드, 필터, 참조선 등에 적용할 수 있는 변수<br>
➡️ "Zone Sales" : 매개변수 기반으로 구성한 변수<br>
